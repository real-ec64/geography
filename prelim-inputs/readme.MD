#### DATA METHODOLOGY

## i2n and n2n dict
Taken from pycountryconverter package cc.data
Ad-hoc changes made as necessary (e.g Türkiye  --> Turkey)

## Country-list
Taken from Icyrockcomo's github csv at github.com/icyrockcom/country-capitals/blob/master/data/country-list.csv
Ad-hoc changes made as necessary

## Gdp and pop
Taken from World Bank most recent data (2024)
Where data not available, either last available date data used or data taken from google search.
https://data.worldbank.org/indicator/SP.POP.TOTL
https://data.worldbank.org/indicator/NY.GDP.MKTP.CD
